### 第三章控制流

前面的章节，我们已经使用过一些控制流语句，例如`if`和`for`，接下来我们将详细介绍这些语句的使用方法。

#### 3.1 语句和分程序

1. C 语言中，分号 `;` 是语句终结符，每个语句必须以分号结束。
2. 可以用花括号 `{}` 来把几个语句构成一个复合语句。

#### 3.2 if 语句系列

1. if 语句最基本的情况如下:

    ```clike
    if (条件)
        语句;
    ```

    如果条件成立，就执行语句，否则跳过语句。

2. 带有 else 的情况如下:

    ```clike
    if (条件)
        语句1;
    else
        语句2;
    ```

    如果条件成立，就执行语句1，否则执行语句2。

3. 带有 else if 的情况如下:

    ```clike
    if (条件1)
        语句1;
    else if (条件2)
        语句2;
    ```

4. 带有 else if 和 else 的情况如下:

    ```clike
    if (条件1)
        语句1;
    else if (条件2)
        语句2;
    else
        语句3;
    ```

    如果条件1成立，就执行语句1，否则如果条件2成立，就执行语句2，否则执行语句3。

上面所有的语法按照实际需要使用即可，都是是合法的。

#### 3.3 switch 语句

switch 语句是一种多分支语句，它可以根据表达式的值，跳转到不同的语句。

switch 语句的基本语法如下:

```clike
switch (表达式)
{
    case 常量表达式1: 语句1;
    case 常量表达式2: 语句2;
    ...
    case 常量表达式n: 语句n;
    default: 语句n+1;
}
```

一般情况下，每个 case 语句后面都要加上 break 语句，否则会出现穿透现象。

```clike
switch (表达式)
{
    case 常量表达式1: 语句1; break;
    case 常量表达式2: 语句2; break;
   ...
    case 常量表达式n: 语句n; break;
    default: 语句n+1; break;
}
```

#### 3.4 while 与 for 循环

1. while 循环的基本语法如下:
    ```clike
    while (条件)
        语句;
    ```
    如果条件成立，就执行语句，否则跳过语句。

    while 语句可以配合 break 和 continue 语句使用，break 语句可以跳出循环，continue 语句可以跳过本次循环。

2. for 循环的基本语法如下:

    ```clike
    for (表达式1; 表达式2; 表达式3)
        语句;
    ```

    同样 for 语句可以配合 break 和 continue 语句使用，break 语句可以跳出循环，continue 语句可以跳过本次循环。

while 配合 break 的例子:

```clike
int i = 0;
while (i < 10)
{
    if (i == 5)
        break;
    printf("%d\n", i);
    i++;
}
```

while 配合 continue 的例子:
```clike
int i = 0;
while (i < 10)
{
    if (i == 5)
        continue;
    printf("%d\n", i);
    i++;
}
```
