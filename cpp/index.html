课程简介

  • 本课程将介绍 C/C++ 语言的基础知识，包括 C/C++ 语言的特性、语法、标准库等。

  • C++ 语言部分将不涉及面向对象的内容，因此大部分的语法与 C 语言相同。

课程目标

  • 掌握 C/C++ 语法后和标准库后，编写简单的命令行程序，如 ls 命令。 

    								
									$ ls
									files ...

  • 通过 C 语言和单片机，实现电子电路设备控制。
