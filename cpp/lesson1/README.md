#### 1. 第一个 C++ 程序

接下来我们将编写一个最简单的 C++ 程序，并尝试编译与运行该程序，完整的体验一下从代码编写到运行的整个过程。

使用前面学习过的 `vim` 编辑器，创建一个名为 `main.cpp` 的文件。

```bash
$ vim main.cpp
```

并输入如下代码：

```cpp
#include <iostream>

using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    
    return 0;
}
```

保存并退出编辑器，然后使用 `g++` 编译程序：

```bash
$ g++ main.cpp
$ ./a.out
```

执行命令后，屏幕上就会显示出 `Hello, World!`。

这就是一个最简单的 C++ 程序的完整开发过程，后面的学习将会慢慢了解上面这个程序的各个部分，现在只需要有一个大致的体验即可。

#### 2. 变量与常量（课本第二章第三节）

##### 2.1 认识变量

软件开发中，会用到很多的数据，这些数据就保存在各种 `变量` 中，所以我们最需要先认识变量。

比如应用题中，小明家到学校的距离是 200 米，小明步行的速度可以记为一个变量 `v`，当 v 为 2 米每秒时, 小明从家到学校的时间是 100 秒, 
当 v 为 5 米每秒时, 小明从家到学校时间是 40 秒。v 作为一个变化的量，可以根据 v 来计算出小明从家到学校的时间。软件开发中也会需要这样的一个量来做各种计算。

##### 2.2 变量的定义

一个变量主要包括两个部分：变量名和变量类型。下面是一个变量的定义：

```clike
int main() {
    // 定义一个整型变量 a，语法为 
    int a;
    
    return 0;
}
```

> 定义变量的语法为 `数据类型 变量名`，除了上面的整型（相当于我们数学里面的自然数 0, 1, 2...），后面还会接触到其他的数据类型。

##### 2.3 变量的赋值

变量的赋值语法为 `变量名 = 值`，比如：

```clike
int main() {
    int v;
    
    v = 10;
    
    return 0;
}
```

上面这段代码的含义是定义一个整型变量 `v`，然后把 10 "装"到 v 中，与数学中的等于号不一样。要注意的是赋值是数据的类型要与变量的类型一致。
比如 v = 0.1 ，小数部分就会丢失。
