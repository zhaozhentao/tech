#### 数据类型(第二章第四节)

##### 整型

C++ 提供了以下 8 种整型类型（数据类型和范围会是考点之一）：

处理不同文件时，可以根据所需要的数据范围来选择合适的整型类型，通常来说如果没有特别要求，简单选用 int 也可以，如果想要更好的内存消耗表现，可以选择空间占用更少的数据类型。

| 数据类型 | 描述 | 大小(字节) | 值范围 |
|---------|------|-----------|--------|
| short | 短整型 | 2 | -32,768 到 32,767 |
| int | 整型 | 4 (通常) | -2,147,483,648 到 2,147,483,647 |
| long | 长整型 | 4 (通常) | -2,147,483,648 到 2,147,483,647 |
| long long | 长长整型 | 8 | -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807 |
| unsigned int | 无符号整型 | 4 (通常) | 0 到 4,294,967,295 |
| unsigned short | 无符号短整型 | 2 | 0 到 65,535 |
| unsigned long | 无符号长整型 | 4 (通常) | 0 到 4,294,967,295 |
| unsigned long long | 无符号长长整型 | 8 | 0 到 18,446,744,073,709,551,615 |

示例，使用上面这几种不同的类型来定义变量。

```clike
int main() {
    // 下面的变量名可以修改为其他，这里只是做演示
    short s;
    int i;
    long l;
    long long ll;
    unsigned int ui;
    unsigned short us;
    unsigned long ul;
    unsigned long long ull;

    return 0;
}
```

##### 浮点型(小数类型)

浮点类型也就是我们日常使用的小数类型，C++ 提供了以下 3 种浮点类型：

| 数据类型 | 描述 | 大小(字节) | 值范围 |
|---------|------|-----------|--------|
| float | 单精度浮点型 | 4 | 3.4E-38 到 3.4E+38，精度约为 6 ~ 7位小数 |
| double | 双精度浮点型 | 8 | 1.7E-308 到 1.7E+308，精度约为 15 ~ 16 位小数 |
| long double | 扩展精度浮点型 | 16 | -3.4E+4932 ~ 1.1E+4932 精度约为 18 ~ 19 位小数 |

> 如果计算涉及小数处理，就需要使用浮点类型变量，如果使用整型变量，会丢失掉小数部分。

##### 字符型

字符类型用于存储单个字符，C++ 提供了 char 类型来表示字符，如下面的例子变量 a 保存了一个字符数据，而且该字符需要用单引号 '' 括气来。

```clike
int main() {
    char a = 'c';

    return 0;
}
```

字符变量其实可以转换为整型，转换的规则是将字符的 ASCII 码值赋值给整型变量。点击查看 [ASCII](https://www.runoob.com/w3cnote/ascii.html) 表。

```clike
#include <iostream>

using namespace std;

int main() {
        char a = 'c';

        int b = a;

        // 把变量 b 输出到屏幕中。
        cout << b << endl;

        return 0;
}
```
