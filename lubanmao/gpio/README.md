##### 引脚号

不论是通过命令行、还是通过代码控制 GPIO，都需要知道 GPIO 引脚的编号，引脚号的计算方式：

引脚编号 = 引脚控制器编号 * 32 + 引脚编号

引脚控制器编号：A = 0, B = 1, C = 2, D = 3, E = 4, F = 5, G = 6, H = 7 ...

引脚号：0 ~ 31

比如引脚 PH7 的引脚号是 7 * 32 + 7 = 231
