#### 指针与数组

C语言中，指针与数组的关系十分密切，通过数组下标所能完成的操作都能使用指针完成，一般来说，用指针编写的程序比用数组下标编写的程序执行速度快，但通过数组下标编写的程序要容易理解一些。

##### 数组元素与地址

C语言中，数组可以理解为相同数据类型的变量的集合，整个数组在内存中占据一片连续的内存空间。所以我们自然也可以取得数组中每个元素在内存中的地址。

```clike
// 数组的定义
int a[10];

// 从数组中取某一个元素
a[i];

// 数组中某个元素的地址
&a[i];
```

假如有一个指向 int 类型的指针 p，那么就可以使用这个指针来完成一些之前通过数组下标的操作。

```clike
// 指向 int 类型的指针
int *p;

// 指向数组中的第 i 个元素(先找到第 i 个元素，然后取出该元素的地址)
p = &a[i];

// 读取或写入
int b = *p; // 这里与 int b = a[i]; 等价

// 写入 1
*p = 1;     // 这里与 a[i] = 1; 等价
```

##### 数组名与指针

另外，C语言中数组名字代表数组的起始地址（例如上面的数组名 a ，就代表 &a[0]），所以有下面的关系

```clike
int *p;

// 通过数组名取出数组起始地址
p = a;

// 与上面的结果相同
p = &a[0];
```

> 注意，数组名 a，不是一个变量，不能像修改指针 p 一样，对 a 进行修改，比如 a = p。

##### 指针的运算

类似与数组通过下标取不同的元素，指针也有相应的操作。

```clike
int a[10];

// 读取第 5 号元素
int c = a[5];

int *p = a;

// 读取第 5 号元素
int d = *(p + 5);
```

> 提示，可以总结出这样的结论，p + xx (xx 是一个整数) 等于 p 指向的地址加上 xx 乘以 p 指向的类型大小，例如 p 是指向 int 的指针，p + 2 就等于 p + 2 * 4 (int 的大小为 4)。
> <br/>我们也可以用 printf("%p", p) 显示出具体地址是多少，来验证一下。

#### 字符指针与函数

##### 字符串与字符指针

字符串常量是一个字符数组，例如:

```clike
"I am a string"
```

前面的学习中，我们使用过了很多次的 printf 来输出一些打印信息，比如下面的。

```clike
printf("hello world\n");
```

像这种 printf 函数，是用什么类型的参数来接收字符串的呢，其实是使用 `char *` 类型，所以上面的打印也可以写成：

```clike
#include <stdio.h>

int main() {
        char *message = "hello world\n";

        printf(message);

        return 0;
}
```

所以，类似于这样的字符串出现在程序中，实际上是通过字符指针访问该字符串的。

##### 字符数组字符串

数组、字符串、指针，这三者之间有一些微妙的关系。
1. 数组名和引用字符串的指针，都是一个地址，而且这个地址不可以改变，比如 int a[10]，无法为 a 设置一个新地址。
2. 创建一个字符数组时，可以用一个字符串来初始化该数组。
3. 数组元素可以改变，字符串无法改变。

可以看看下面例子。
```clike
#include <stdio.h>

int main() {
        /*
         * 这种数组创建方式，实际上应该理解为 char a[] = {'h', 'e', 'l', 'l', 'o', ' ', 'w', 'o', 'r', 'l', 'd', '\n'};
         * 这样可以更直观的与 char *p = "hello world\n" 区分开来。
         */
        char a[] = "hello world\n";

        printf(a);

        // 数组元素可以改变
        a[0] = 'H';

        // 输出改变后的内容
        printf(a);

        char *p = "hello world\n";

        printf(p);

        // 执行这一行时，会崩溃
        *p = 'H';

        printf(p);

        return 0;
}
```

> 字符串总是以 '\0' 结尾，所以程序可以通过检查空字符找到字符串的结尾，字符串常量占据的存储单元数比双引号内的字符数大 1 ，比如上面的字符串 "hello world\n"，实际上长度是 13。

```clike
#include <stdio.h>

int main() {
        char a[] = "hello world\n";

        printf("%ld\n", sizeof(a));

        printf("%ld\n", sizeof("hi"));

        return 0;
}
```
