### 低代码平台生成表单 (CRUD)

下面通过创建一个 `开发客户` 记录表来说明一下 **云程** 低代码平台 (CRUD) 的能力，主要包括下面几个步骤。

1. 创建应用
2. 创建数据实体（数据库表）
3. 构建数据实体的创建表单（对应 CRUD 的增加动作）
4. 构建数据的查询表格（包含 CRUD 的查询、编辑、删除动作）

> 低代码开发一般有单纯的生成前端表单代码，以及像云程一样的商业系统，通过可视化方式完成前端后台需要的完整功能，前者只是减少了前端写页面的时间，仍然需要开发人员手动编写各种业务代码。

#### 3 构建数据实体的创建表单

##### 1. 构建创建数据的表单

有了数据库表就可以创建相应表单，用于实现相关数据的新增操作。

上一步创建好数据实体后，已经自动创建了默认的表单，在此基础上进行 `PC 表单设计`。

![img.png](images/low/form1.png)

使用默认模板，自动添加了在设计 `数据实体` 时自己添加的字段表单项，保存表单。

![img.png](images/low/form2.png)

##### 2. 添加表单访问入口

上一步，创建了一个 `添加数据` 的表单，需要在 `菜单` 中添加访问入口来访问刚刚创建的表单。

![img.png](images/low/add_menu.png)

![img.png](images/low/add_menu2.png)

##### 3. 设置表单浏览权限

访问入口添加完成后，还需要设置表单的浏览权限，否则无法访问到表单。

![img.png](images/low/add_menu3.png)

##### 4. 访问表单

添加好权限后，可以通过云程的前台访问到表单。在里面填写好表单数据，点击 `保存` 按钮，即可完成数据的新增。
