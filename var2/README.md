#### 2.6 关系运算符与逻辑运算符

##### 2.6.1 基本概念

C 语言中关系运算符包括：
1. `==` 等于
2. `!=` 不等于
3. `>` 大于
4. `<` 小于
5. `>=` 大于等于
6. `<=` 小于等于

C 语言中逻辑运算符包括：
1. `&&` 逻辑与
2. `||` 逻辑或
3. `!` 逻辑非

经过关系运算符和逻辑运算符的运算，会得到一个布尔值，即 `true` 或 `false`，但 c 语言的基本类型中没有布尔类型，
所以我们用 `int` 来表示布尔值，`true` 用 `1` 表示，`false` 用 `0` 表示，可以用 `非 0 即真` 来记忆。

```clike
int a = 1;
int b = 2;
int c = a > b; // c 的值为 0，即 false
int d = a < b; // d 的值为 1，即 true
```

##### 2.6.2 短路运算

逻辑运算符有短路运算的特性，即当第一个条件已经可以确定结果时，后面的条件就不会再判断了。

```clike
int a = 1;
int b = 2;
int c = 3;
int d = (a > b) && (c > b); // 因为 (a > b) 为 false，所以后面的 (c > b) 不会再判断了，d 的值为 0，即 false
```

思考，如何通过代码验证上面说的例子？

> || 的运算也有类似的短路运算特性，你能举个例子吗。

##### 2.6.3 优先级

运算符有优先级（具体优先级可以参考书本），就像数学中的乘除加减一样， 1 + 2 * 3 的结果是 7。

实际使用中，我们可以使用括号来明确优先级，如下面例子，这样可以减少阅读代码时关于优先级的困惑。

```clike
int c = (1 > 2) && (3 < 4)
```

#### 2.7 类型转换

当一个表达式中包含不同类型的变量时，会发生类型转换，把操作数转换为同一种类型，然后再进行运算。一般来说自动转换是指把“比较窄的”操作数转换为“比较宽的”操作数（精度不会丢失），如把 `int` 转换为 `float`（如果反过来，float 转换成 int 精度就丢失了）。

```clike
int a = 1;
float b = 2.0;
float c = a + b; // c 的值为 3.0
```

> 练习，尝试使用一下书本中的 isdigit 函数，判断一个字符是否为数字。

##### 隐式转换

上面提及到的自动转换就是一种隐式转换，详细的转换规则可以参考书本。

> 赋值时，右边的值会被转换为左边变量的类型。

##### 显式转换

显式转换是指把“比较宽的”操作数转换为“比较窄的”操作数，如把 `float` 转换为 `int`。

```clike
float a = 1.1;
int b = (int) a; // b 的值为 1
```

> 注意！这里不会改变 a 的值，只是 “临时” 把 a 的值转换为 int 类型，然后赋值给 b。

#### 2.8 自增运算符和自减运算符

自增运算符 `++` 和自减运算符 `--` 是一种特殊的运算符，它们可以在变量前面或后面使用，分别表示自增 1 和自减 1。

搞明白下面，代码执行结果。

```clike
int a = 1;
int b = a++; // 先将 a 原来的值赋值给 b，然后 a 自增 1
int c = ++a; // 首先 a 已经是 2，然后 a 自增 1，然后将 a 自增 1 后的值赋值给 c
```
