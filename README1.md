### 1.3 For 循环

C 语言中的 for 循环是一种非常常用的循环结构，它允许代码块被重复执行指定的次数。以下是for循环的基本语法和使用方法：

```clike
for (初始化; 条件; 更新) {
    // 循环体
} 
```

* 初始化：在循环开始前执行一次，通常用于定义和初始化循环变量。
* 条件：在每次循环迭代前检查。如果条件为真（非零），则执行循环体；如果条件为假（零），则终止循环。
* 更新：在每次循环体执行完毕后执行，通常用于更新循环变量。

#### 示例

上一节，我们用 while 循环实现了 1 到 10 人民币兑换成美金的转换。如果使用 for 循环，可以如下所示：

```clike
#include <stdio.h>

int main() {
        int rmb;

        // 初始化部分: 声明变量 dollar 并初始化成 1
        // 条件: for 循环执行条件为 dollar <= 10 (即 dollar 大于 11 时，退出 for 循环)
        // 更新: 每次循环体执行完毕后，dollar 的值都会加 1 ，然后执行 `条件` 判断是否继续执行括号中代码。
        for (int dollar = 1; dollar <= 10; dollar++) {
                rmb = dollar * 7;
                
                printf("%d 美元可以兑换 %d 人民币\n", dollar, rmb);
        }

        return 0;
}
```

在实际编程过程中，根据问题的具体需求和编程风格选择 while 或 for 循环结构，以实现更高效的代码编写。

### 1.4 符号常量

在 C 语言中，符号常量是通过 `#define` 预处理器指令定义的标识符，它们代表固定的值，在程序编译前会被替换为实际的值。符号常量可以提高代码的可读性和维护性。

以下是一个示例：

```clike
#include <stdio.h>

#define PI 3.14159265358979323846

int main() {
   double radius = 5.0;

   // 计算圆的面积
   double area = PI * radius * radius;
   
   printf("圆的面积为：%.2f\n", area);
}
```

在上面的例子中，`#define` 指令定义了一个符号常量 `PI`，它的值为 `3.14159265358979323846`。这个常量可以用在任何需要一个固定值的地方。

### 1.5 字符输入/输出

在 C 语言中，字符输入/输出是通过 `getchar()` 和 `putchar()` 函数实现的。`getchar()` 函数用于从标准输入设备（键盘）读取一个字符，而 `putchar()` 函数用于向标准输出设备（屏幕）写入一个字符。

下面的例子读取键盘输入的字符，并输出到屏幕上。

```clike
#include <stdio.h>

int main() {
        char c;

        while ((c = getchar()) != EOF) {
                putchar(c);
        }

        return 0;
}
```
